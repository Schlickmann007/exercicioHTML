HTML

Definição

HTML é a sigla para HyperText Markup Language, em portugues Linguagem de Marcação de Hipertexto.
É ultilizada para criar paginas para Web através de marcações no conteudo do arquivo chamado de tags.

Histórico

1.HTML foi criado por tim berners-lee em 1980 baseado no conseito de hypertexto. inicialmente desenvolvido em pascal.

2.Tim Bernes- Lee e o estudante Robert Cailliau implementaram a primeira comunicação bem sucedida entre um cliente HTTP e um servidor através da internet.

3. O HTML surge oficialmente em 1990.
